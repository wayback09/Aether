Aether

A minimal, extensible, lightning-fast Minecraft launcher.

This launcher exists to launch Minecraft, and nothing more. Every non-essential feature is an extension. Think of it as the VS Code of Minecraft launchers.

Features

Blazing Fast

Cold starts in under 2 seconds. Idle memory footprint below 100MB. Navigation is instant. Aether is built to feel like a native system utility, not another Electron app.

Minimalist UI

Clean, native-feeling design with a frameless window. Dark mode first. Inspired by VS Code, Linear, Raycast, and Notion. The UI never gets in your way.

Extensible Architecture

Everything beyond launching Minecraft is an extension. Mod loaders, mod browsers, server lists -- all pluggable. Extensions run in isolated Goja JavaScript sandboxes with a rich API.

Instance Management

Create, rename, delete, import, and export Minecraft instances. Each instance has its own version, mod loader, and memory allocation. Switch between setups without any hassle.

Secure Sandbox

Extensions run in strict capability-based JavaScript isolates. They cannot access your filesystem, OS, authentication tokens, or other extensions. Trust is explicit and granular.

Automatic Java Management

Detects, downloads, and manages Java runtimes from Adoptium. Required Java version is auto-detected based on your Minecraft version. No manual setup needed.

Microsoft OAuth2

Full Microsoft / Xbox / XSTS / Minecraft authentication chain. Also supports offline accounts for development and testing.

Command Palette

Ctrl+Shift+P command palette for quick navigation and actions. Jump to any page, create instances, or toggle settings without leaving the keyboard.

Extension System

Aether's power comes from its extensions. The core stays lean while the community builds on top. Extensions can register sidebar pages, add instance types, hook into the launch pipeline, and communicate with the UI via a secure IPC bridge.

Built-in Extensions

Fabric -- Install and manage the Fabric mod loader for any instance. Handles classpath injection and version resolution automatically.

Modrinth Browser -- Browse and install mods from Modrinth directly into your instances. Search, filter, and install with a clean interface.

For Developers

Extensions run in Goja JavaScript sandboxes with a well-documented API. The Aether global object provides access to instances, extensions, UI, auth, and Java subsystems. Extensions are packaged as ZIP files with a manifest and can be distributed through the live Extension Gallery.

Aether.instances Aether.extensions Aether.ui Aether.auth Aether.java
